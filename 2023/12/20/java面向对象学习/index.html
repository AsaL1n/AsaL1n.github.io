6. final

修饰类 类就不能被继承了

修饰方法，方法是最终方法，也不能被重写了

修饰变量，变量只能被修饰一次

基本类型 的变量是不可以去修改的

引用性的变量不能修改地址但是可以修改地址所指向的值

static final

类似c里面的define 定义一个常量

需要大写加下划线构建

程序在编译的时候会直接进行宏替换，与原来的时候是一样的
