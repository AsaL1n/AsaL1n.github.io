Zip Slip是一种在压缩包中特制(../../../evil.sh)的解压缩文件替换漏洞，包括多种解压缩如tar、jar、war、cpio、apk、rar、7z和zip等。

后端使用解压类直接将压缩包当中的节点解压出来，可能会通过节点的名字../跳转到上级目录中，从而导致任意目录的文件替换
